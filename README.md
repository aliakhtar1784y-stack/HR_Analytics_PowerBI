HR Analytics Dashboard Power BI

 Project Overview
This project focuses on analysing employee attrition using an interactive Power BI dashboard**.  
The dashboard helps HR and business stakeholders understand key factors influencing attrition such as **job role, salary, age, education, gender, and job satisfaction**.


 Business Questions Addressed
- What is the overall employee attrition rate?
- Which job roles experience the highest attrition?
- How does salary band impact attrition?
- Which age group is most affected by attrition?
- How does job satisfaction vary across roles?

 Tools & Technologies
- **Power BI Desktop**
- **DAX** (measures & calculations)
- **Excel / CSV dataset**
- **Data Cleaning & Modelling**



##  Key Metrics
- Total Employees: **1413**
- Total Attrition: **229**
- Attrition Rate: **16.21%**
- Average Age: **37**
- Average Years at Company: **7**
- Average Salary: **6.5K**
